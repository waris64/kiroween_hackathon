### Common Questions

**Q: Does it work with private repos?**
A: No, only public repositories for security reasons.

**Q: How long does analysis take?**
A: 30-60 seconds for small repos, longer for large ones.

**Q: What if AI fails?**
A: We have fallback text that still looks great!

**Q: Can I analyze any Git host?**
A: GitHub, GitLab, and Bitbucket are supported.

**Q: What's the tech stack?**
A: React, Node.js, Google Gemini AI, Framer Motion, TailwindCSS.
